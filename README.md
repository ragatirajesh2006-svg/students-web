# Student Management System

This is a web-based Student Management System built using Python (Flask).

## Features
- Student Login
- Teacher Login
- Attendance Management
- Marks Management
- Admin Dashboard

## Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite / MySQL

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run app:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
